# Jake - To Do List

Jake é um projeto simples de **lista de tarefas (To Do List)** desenvolvido com **HTML, CSS e JavaScript**, criado com o objetivo de praticar e demonstrar os conceitos fundamentais do desenvolvimento web.

O projeto permite ao usuário **adicionar, visualizar e remover tarefas**, oferecendo uma interface simples e funcional para organização de atividades do dia a dia.

## Tecnologias Utilizadas

- **HTML5**  
  Responsável pela estrutura da aplicação e organização dos elementos da página.

- **CSS3**  
  Utilizado para estilização da interface, proporcionando um layout mais agradável e intuitivo.

- **JavaScript**  
  Responsável pela lógica da aplicação e pela interatividade, permitindo a manipulação dinâmica da lista de tarefas.

## Funcionalidades

- Adicionar novas tarefas
- Remover tarefas da lista
- Interface simples e intuitiva
- Atualização dinâmica sem recarregar a página
